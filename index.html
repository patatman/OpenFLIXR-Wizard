OpenFLIXR is incredibly feature rich. Therefore applications are preconfigured with OpenFLIXR defaults. There are just a few steps to go through to get your own OpenFLIXR up and running!

Before you can use OpenFLIXR, you need to mount ‘download’ (subdirectories will be created at boot time when mounted to a network share), ‘movies’, ‘series’, ’music’ and ‘comics’ folders from a location with enough free disk space like a NAS, external harddrive, etc.

To make things easier you can use Webmin to accomplish this. OpenFLIXR is case sensitive.

To be able to use OpenFLIXR with default settings, you must use the following directory structure!

/mnt/downloads/
      blackholenzb
      blackholenzbget
      complete
      incomplete
      music
      musicready
      temp
      tempcomics
      tempseries
      torrentcomplete
      torrentwatch
