Let's go!

OpenFLIXR is incredibly feature rich. To make it easier to get started, applications are preconfigured with OpenFLIXR defaults. There are just a few steps to go through to get your own OpenFLIXR up and running!

This setup should be performed as soon as you’re done reading. It will take care of the following:


Ask for a user defined password
Guide you through the basic stuff like network, Newznab and Usenet settings
Setup Let’s Encrypt
Generate random API keys
Install Plex Media Server (to comply with TOS)
